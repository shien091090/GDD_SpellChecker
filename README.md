## 簡介
- 遊戲標題：鑒魔士菲莉娜
- 遊戲類型：文字冒險+解謎+益智
- 目標平台：PC(Steam)
- USP：像「威力在哪裡」一樣從豐富而充滿細節的俯瞰地圖中尋找推進劇情的蛛絲馬跡、蒐集線索, 為魔法詐騙案的受害者拼湊出完整的犯罪經過, 劇情推進的過程中會有許多解謎小遊戲, 讓整體遊戲體驗更具趣味性

## 背景設定
在魔法普及的世界中, 偶爾會有一些奇特案件並非由魔法師所為, 而是巧妙地利用機關、障眼法、魔術伎倆, 刻意布置的像是「魔法犯罪」, 這類手段背後隱藏著兩大原因, 其一是鑽法律漏洞。牽涉到魔法的犯罪案件必須交由「魔法審判所」審訊, 當罪犯被拘捕時無法當下定罪, 而是需要押去路途遙遠的審判所, 這給了犯罪組織轉移犯人的空檔, 被收買的護衛常以「魔物襲擊」或「山賊攔路」為由放走囚犯。儘管此類伎倆早已是公開的秘密, 但在人送達審判所前, 官方無法定罪, 地方官員也多因收受賄賂或無力管轄而睜一隻眼閉一隻眼。其二是利用民眾對強權的恐懼, 魔法師往往擁有強大武力或是出身顯赫, 即使是知情者也會畏懼規模龐大的犯罪組織。即便如此, 依然有少數渴求政績的新任官員, 或是實力雄厚的冒險者公會試圖撼動這些犯罪組織, 「鑒魔士」這一職業由此而生。
鑒魔士是通曉萬物的雜學家, 其知識橫跨魔法學、製藥學、化學、魔術戲法與機械工程。在試圖鑿穿犯罪組織那道由龐大利益築成的厚牆時, 他們無疑是罪犯們的眼中釘。由於經常深入險境, 強大的自衛能力成了這項職業的必備條件。本作的故事便圍繞著鑒魔士菲莉娜與她的護衛札隆克爾展開。
菲莉娜, 種族為人類女性, 出生於北方穀倉「蛙掌城」, 童年的回憶總與眼鏡、書本以及通往圖書館的小徑掛鉤, 家境雖平庸, 父母仍竭盡所能將這名聰慧的女孩送往學術聖殿-賢者之城「葛洛夫利迪」上頂尖大學, 然而一場精心設計的騙局毀了一切。她在求學期間誤信一名偽裝成真命天子的犯罪組織成員, 不僅被奪走初夜, 更被騙光了所有積蓄。即便父母典當家產也難以支撐她完成學業, 一夕之間夢想、尊嚴、愛情全部化為泡影, 從此她便下定決心未來走上打擊犯罪之路, 在工作了多年之後, 她存夠了錢回到葛洛夫利迪完成學業並取得文憑, 隨後在未來順利得到了「鑒魔士」的職位。
札隆克爾, 種族為人類男性, 和菲莉娜是老鄰居, 與書堆中長大的菲莉娜不同, 他自幼便在軍營打雜與市集搬運中磨練, 這練就了他壯碩如山的身形, 雖然外表兇惡但實際上個性單純憨厚。十五歲那年, 母親因誤信假冒魔法師的偏方, 長期服用實為慢性毒藥的藥水, 導致身體崩潰、臥病在床, 市場攤販的經營與照顧母親的重擔落到了札隆克爾肩上, 在形形色色的人們所來往的市集中工作, 單純的札隆克爾也被迫要去學習洞察人心, 也漸漸明白如何善用自己天生令人畏懼的兇惡臉孔。
兩人的重逢始於一場市集的偶遇, 菲莉娜在就任鑒魔士前正為尋找可靠夥伴發愁, 而渴望改善家人生活的札隆克爾則需要一份高薪工作, 兩人在幾次促膝長談後正式結成了搭檔, 從此展開了在各種窮凶惡極的犯罪案件之中遊走的鑒魔士職涯。

## 核心玩法
分為多個篇章, 每一個篇章的背景設定是通用的, 可能會有少許劇情貫通, 但基本上都是獨立故事篇章。
玩家可以在由數張區塊所構成的地圖上移動, 地圖類似「威力在哪裡」豐富而充滿細節, 移動方式及其他任何操作都僅透過滑鼠點擊完成, 人物並不會在地圖上有移動的動畫, 頂多就是跑劇情時會在地圖上出現角色靜止圖片。
章節開始時會以劇情演出的方式告訴玩家此次案件從何開始, 隨著受害人自白, 會列出幾個待調查的疑點, 劇情演出的形式為立繪加上對話框, 接著玩家就要去尋找線索, 把所有疑點調查清楚, 找到關鍵證據, 鎖定罪犯, 將他們繩之以法。
主要流程大致可以分為幾個步驟：
1.跑劇情, 通常以委託人的自述開場。
2.蒐集線索, 玩家可在不同地圖中移動尋找蛛絲馬跡。
3.每次蒐集到必要線索, 啟動小遊戲, 獲得推進劇情的關鍵要素。
4.跑下一段劇情, 重複1~3, 會有系統提示玩家目前劇情推動進度。
5.劇情碎片拼湊完畢, 還原完整犯案流程, 將罪犯繩之以法, 跑Ending劇情。
