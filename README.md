## 簡介
- 遊戲標題：魔法詐騙鑑定士
- 遊戲類型：文字冒險+解謎+益智
- 目標平台：PC(Steam)
- USP：像「威力在哪裡」一樣從豐富而充滿細節的俯瞰地圖中尋找推進劇情的蛛絲馬跡、蒐集線索, 為魔法詐騙案的受害者拼湊出完整的犯罪經過, 劇情推進的過程中會有許多解謎小遊戲, 讓整體遊戲體驗更具趣味性

## 背景設定
在一個存在魔法的世界中, 時不時會有一些犯罪事件並不是由魔法師所為, 而是使用了機關、障眼法、魔術伎倆等手段布置的像「魔法犯罪」, 罪犯們這麼做的原因有幾個, 其一, 牽涉到魔法的犯罪必須由特殊的政府機關「魔法審判所」來審訊, 所以當罪犯被拘捕時, 無法當下定罪, 而被押去魔法審判所的路上通常會由馬車乘載, 伴隨護衛數名前往, 然而大多這類型犯罪都是由比較有規模的犯罪組織所為, 早就把護衛都買通了, 在半路上他們就會被悄悄地釋放, 偽裝成「魔物襲擊」或「山賊襲擊」等理由, 這種犯罪流程路人皆知, 但因為人還沒送到審判所, 所以審判所方並不會為此買單, 而當地官員要不是收了好處, 就是無力管轄。其二, 魔法師通常擁有強大的力量, 或是出身顯赫貴族, 一般百姓不敢招惹。雖說如此, 但總是會有想拿出一番作為, 端出漂亮政績的新任官員, 或是保障很好、後台夠硬的冒險者公會想去試圖打擊這些犯罪組織, 「魔法詐騙鑑定士」這一職業由此而生。

## 核心玩法
分為多個篇章, 每一個篇章的背景設定是通用的, 可能會有少許劇情貫通, 但基本上都是獨立故事篇章。玩家可以在幾張地圖上移動, 地圖類似「威力在哪裡」一樣豐富而充滿細節, 移動方式及其他任何操作都僅透過滑鼠點擊完成。劇情演出的形式為立繪加上對話框, 人物並不會在地圖上有移動的動畫, 頂多就是跑劇情時會在地圖上出現角色靜止圖片。
